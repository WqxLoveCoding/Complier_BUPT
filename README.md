### 2、数据结构设计：（符号表、语法树、记号二元组...）  

**符号表**

| 名称 | 类型 | 维度 | 声明行 | 指针域 |
| ---- | ---- | ---- | ------ | ------ |
|      |      |      |        |        |

1. 名称：标识符的具体名称（字符串）
2. 类型：整型，实型等基本类型（也可以是数组的类型，函数返回值类型）
3. 维度：数组维度，如果是基本类型则为0
4. 声明行：标识符首次声明的位置
5. 指针域：函数和过程的子表指针

（函数/过程类型通过指针+类型判断及区分，数组类型通过维度区分）
